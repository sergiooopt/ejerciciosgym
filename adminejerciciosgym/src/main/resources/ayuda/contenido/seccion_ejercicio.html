Sección Ejercicio

Panel donde se introducen y editan los datos básicos del ejercicio.

Campos

Nombre
Texto que identifica el ejercicio (ej: "Press banca", "Sentadilla"). Campo obligatorio.
Descripción
Descripción breve del ejercicio. Campo obligatorio.
Imagen
Ruta absoluta de la imagen representativa del ejercicio en el servidor. Debe seleccionarse Elegir imagen y elegir la imagen deseada respecto al ejercicio. Cuando se guarda el formulario la imagen se guarda usando ruta relativa en el servidor.
Peso mínimo
Valor numérico entero que representa el peso mínimo recomendado para el ejercicio.
Peso máximo
Valor numérico entero que representa el peso máximo recomendado para el ejercicio.

Botones

Elegir imagen
Abre un explorador de archivos para seleccionar.jpg o .jpeg. Es necesario para la validación haber seleccionado una imagen.
Guardar formulario
Guarda el ejercicio junto con todos los músculos involucrados. Si se selecciona un ejercicio existente en la tabla (doble clic), lo actualiza. Si no hay selección previa, crea un ejercicio nuevo. Valida que todos los campos obligatorios estén rellenos y que la imagen esté validada.
